# 画面設計

## 画面一覧

| 画面 | パス | 認証 | 説明 |
|------|------|------|------|
| ログイン | /login | 不要 | メール+パスワードでログイン |
| サインアップ | /signup | 不要 | 新規アカウント作成 |
| 在庫一覧 | / | 必要 | 品目をカード形式で一覧表示 |
| 品目登録 | /items/new | 必要 | 新規品目の登録フォーム |
| 品目編集 | /items/[id]/edit | 必要 | 既存品目の編集フォーム |

---

### 在庫一覧画面 `/`（ホーム）

```
┌─────────────────────────────────┐
│  日用品在庫管理          [+]   │
├─────────────────────────────────┤
│ ┌─────────────────────────────┐ │
│ │ シャンプー 詰め替え         │ │
│ │ YOLU...    [－] 2個 [＋] [⋮]│ │
│ └─────────────────────────────┘ │
│ ┌─────────────────────────────┐ │
│ │ 天然水 500ml                │ │
│ │ amazon     [－] 5本 [＋] [⋮]│ │
│ └─────────────────────────────┘ │
│ ┌─────────────────────────────┐ │
│ │ 洗顔                        │ │
│ │ パーフェ...[－] 0個⚠️[＋][⋮]│ │
│ └─────────────────────────────┘ │
│ ┌─────────────────────────────┐ │
│ │ リステリン 1500ml           │ │
│ │ トータル...[－] 1本 [＋] [⋮]│ │
│ └─────────────────────────────┘ │
│ ┌─────────────────────────────┐ │
│ │ コンタクト 2weeks           │ │
│ │ シード..   [－] 1箱 [＋] [⋮]│ │
│ └─────────────────────────────┘ │
└─────────────────────────────────┘

レイアウト:
- 1行目: 品目名
- 2行目: 商品名(省略) + [-]ボタン + 在庫数+単位 + [+]ボタン + [⋮]メニュー
- カード高さ: 約60px（2行構成）
- スマホ1画面で5-6件表示可能

[⋮] メニュー: 編集 / 削除
[+] 品目追加ボタン（ヘッダー右）
⚠️ 在庫0の場合に警告表示（数字の横）
```

### 品目編集画面 `/items/[id]/edit`

- 品目登録画面と同じレイアウト
- 既存データが入力済み
- ボタンは「更新する」

## コンポーネント構成

```
app/
├── (auth)/
│   ├── login/page.tsx
│   └── signup/page.tsx
├── (main)/
│   ├── layout.tsx (認証チェック)
│   ├── page.tsx (在庫一覧)
│   └── items/
│       ├── new/page.tsx
│       └── [id]/
│           └── edit/page.tsx
└── layout.tsx

components/
├── ui/ (shadcn/ui)
├── item-card.tsx (品目カード)
├── item-form.tsx (品目フォーム)
└── header.tsx (ヘッダー)
```
